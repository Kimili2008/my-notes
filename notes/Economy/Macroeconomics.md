# GDP (Gross Domestic Production)

一个经济体在某一特定时间内按照市场价格所生产的总最终产品价值，通常等于国民总收入
- 最终
- 市场价格
- 特定时间
real GDP：不包含价格变动，常被用来测量居民生活水准，要取某一年的市场价格作为基准
nominal GDP ：包含价格变动
inflation index：通胀指数，nominal/real GDP，检测价格的变动程度

# CPI (Consumer Price Index)

CPI 消费物价指数，衡量生活费用通货膨胀的一个指标
标准定义：
在某段特定时间内，消费者生活所需消费和以前特定一年比较的变化率
普通消费者所购买的物品与服务的总费用的衡量指标
CPI主要组成：食品，交通，住房，食物和饮料，教育和通信
CPI = 某年一basket物品的价格/基年一basket物品的价格

CPI的局限性
1. 替代问题：低估通货膨胀
2. 引进问题：当消费者拥有更多选择，同样一美元的价值是提升的
3. 消费降级问题：当消费长期萎靡不振，教育，通信所需费用会下降，这实际上低估了通货膨胀

Real interest : Nominal Interest - Inflation Index

# 金融市场

金融市场：储蓄者借以直接向借款者提供资金的金融机构

债券：一种债务证明书

债券的特点：
- 期限
- 金融风险
- 税收待遇（除了政府发行的债券，其他都为应纳税收入）

股票市场：企业的所有权

出售股票来筹集资金：权益融资

出售债券来筹集资金：债务融资

公司在股票持有者得到补偿前，优先补偿债券，所以股票持有者有潜在的高风险和高收益

*股票的价格随着需求而变化，需求越高价格越高*

股票指数检测股票的整体水准：如道琼斯指数：选取美国最主要的三十家公司

如标准普尔指数：根据500家主要公司的股票来计算

如何看股票：
- 价格
- 股息（是公司把他的利润支付出的部分显示为股票价格的百分比，没支付的叫留存收益）
- 价格-收益比 P/E ，股票价格/每股收益

官方失业率的局限性
对于失业人数和非劳动力人数的统计是困难的

有些非劳动力和就业者会称自己为失业者，从而得到补助或者避免纳税，这些人并非失业者

有些非劳动力实际上是失业者，他们是discouraged workers，只是由于一直找不到工作才停止求职

*国家通常对于失业人员有安排救济和再就业服务，非劳动力则由于没有就业意愿则无法接受该服务*

- 结构性失业：劳动市场可提供的岗位少于需求岗位（由于工资导致的供需不平衡）
（导致供需不平衡的三个原因：工会，最低工资法和效率工资）

# 工会的作用

工会起到了提高最低工资的作用，随着最低工资的提高，劳动供给量和需求量有差值，这就是失业的劳动力部分，所以工会实际上提升了一部分工人的待遇，同时导致了另一部分工人的失业。

工会对于经济是好是坏？
说法不一，可以确定的是没有工会和各类社会保障之后失业率会降低，但这是一种不健康的提升：公民被迫为了生计去工作赚钱，减少了创新工作的产生。

# 效率工资理论

该理论认为，企业保持高工资对企业本身是好事，也就说明了为何存在结构性失业是必然现象。、
高工资带来：在就业方面的筛选机制，工人群体素质变高，努力工作的激励机制，工人群体更健康
如福特汽车生产线的5美元工资就能说明这一点。


# 古典通货膨胀理论
超速通货膨胀：短时间内大幅通货膨胀（津巴布韦当时的通胀率达到24000%）
温和通货膨胀

### 推导过程

1. 钱币流通速度不变 （流通速度指的是平均每单位货币参与名义GDP的程度，即nominal GDP/ 货币总量）
2. 若政府增加货币总量，短期内名义GDP上升
3. CPI等货量取决于技术，短期内不会大幅提升，变化主要取决于货价
4. 货架上升，每单位货币的购买力下降

古典二分法：区分真实变量和名义变量
真实变量：按实物单位衡量的价值
名义变量：按货币单位衡量的价值
货币价值就是名义变量，真实变量就是不同物品之间的相对价格
1.当物品的价格上升时，货币的价值下降，反之亦然
2.货币的需求量总会等于其供给量

得出结论：物品价值越高，对货币的需求量越大，越低，越小

### 杠杆
leverage：将借到的货币追加到用于投资的现有基金上

所有者权益（资本）的价值等于银行资产（贷款加准备金加有价证券）的价值减负债（存款加上债务）的价值。
leverage ratio：总资产与银行资本的比率，上面的例子中，杠杆率为20，1000/50.
如果银行的资产价值上升了5%，则资本能够上升100%，但下降5%，资本也会下降100%。

### 美联储如何控制货币
公开市场操作：买卖债券
给银行贷款：增加准备金，这种贷款的利率叫贴现率
法定准备金：控制准备金量
支付准备金利息：控制准备金量
联邦基金利率：银行相互的短期贷款，通常只有一夜，通过调整贴现率可以调整基金利率。
